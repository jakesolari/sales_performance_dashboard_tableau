# sales_performance_dashboard_tableau
An interactive sales dashboard analyzing 10,000+ transactions from the 
Tableau Superstore dataset across regions, product categories, and 
sub-categories.

## Live Dashboard
[View on Tableau Public](https://public.tableau.com/views/SalesPerformanceDashboard-FY2023-2026/SalesPerformanceDashboard-FY2023-2026?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

## Project Overview
Built two versions of this dashboard to demonstrate proficiency across 
multiple BI tools — Microsoft Excel and Tableau Public. Both dashboards 
analyze the same dataset and surface the same core business insights.

## Tools Used
- Microsoft Excel — PivotTables, PivotCharts, data formatting
- Tableau Public — interactive visualizations, calculated fields

## Dataset
- Source: Tableau Superstore Sample Dataset
- Rows: 10,194 transactions
- Date Range: FY2023–2026
- Fields: Order Date, Region, Category, Sub-Category, Sales, 
  Profit, Quantity, Discount

## Dashboards & Visuals
- KPI Scorecard — Total Sales, Profit, Orders, Quantity, Profit Margin
- Monthly Revenue Trend (line chart)
- Sales by Region (bar chart)
- Sales by Category (pie chart)
- Top 10 Sub-Categories by Profit (horizontal bar chart)

## Key Findings
- West and East regions account for 60%+ of total revenue
- Technology is the highest grossing category at $840K
- Copiers are the most profitable sub-category at $56K
- Overall profit margin is 12.56% across all categories
- Revenue shows consistent year-over-year growth from 2023–2026
